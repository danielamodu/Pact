# Brag Plan: Pact Protocol

## What is this app?
Pact Protocol is a programmable, cross-chain recurring payment and subscription engine powered by EIP-7702 account delegation and Particle Network Universal Accounts.

## The angle
"Crypto subscriptions always sucked — until EIP-7702."
Showcase the contrast between manual monthly wallet signing and Pact's zero-friction, autonomous session-key subscriptions across Arbitrum and Base mainnets.

## Hook (first 2-3 seconds)
"Subscriptions on Web3. Finally autonomous."
Sleek dark forest green typography popping against mint accents.

## Key moments (the middle)
- **EIP-7702 Delegation**: Granting cryptographically limited session key permissions with hard spending caps.
- **Unified Multi-Chain Mainnet**: Live deployments on Arbitrum One & Base Mainnet (`0x9Db4207...`).
- **Autonomous Settlement**: 1-click execution with on-chain 1% protocol fee split & instant receipts.

## Outro / punchline
"Pact. Set it. Forget it. On-chain."

## User flow worth showing
`Landing Hero` → `Merchant Plan Setup` → `EIP-712 Session Authorization` → `Dashboard Settle & Audit`
